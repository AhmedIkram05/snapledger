## How to Generate Icons

You can use online tools to generate PWA icons from a single image:

1. **PWA Asset Generator**: https://www.pwabuilder.com/imageGenerator
2. **Favicon Generator**: https://realfavicongenerator.net/
3. **PWA Icon Generator**: https://tools.crawlink.com/tools/pwa-icon-generator/

## Design Guidelines

- Use a simple, recognizable icon
- Recommended: A dollar sign ($) or piggy bank icon with a gradient background
- Colors: Use the app's primary colors (#4F46E5, #10B981)
- Ensure the icon works well at all sizes
- Add padding around the icon (safe area)
- Export in PNG format with transparency
